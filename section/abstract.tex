
Modern railway systems require sophisticated remote control capabilities to ensure safe and efficient train operations. In this paper, we implement a remote control system for real-time train operations using WebTransport, a cutting-edge web protocol that combines the reliability of TCP with the low-latency benefits of UDP through HTTP/3 and QUIC. We design a remote control platform that leverages WebTransport's bidirectional streaming capabilities to enable real-time communication between control centers and trains. The proposed system is a working open-source solution that successfully handles concurrent control of multiple trains while maintaining data integrity. Our results indicate that WebTransport-based remote control systems can enhance railway operational efficiency.
