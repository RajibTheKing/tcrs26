\section{System Architecture}
\label{sec:system-architecture}
	This section presents an overview of the system architecture, emphasizing its core components and the interactions among them. Figure~\ref{fig:overview} illustrates the high-level structure of the system, highlighting the main modules and their interconnections. Both the trains and other system components can utilize various communication media such as 4G, 5G, or Wi-Fi, depending on network availability and operational requirements. Central server is responsible to relay the information between the train and remote control client, deployed on a cloud instance, which can be accessed by both through the internet. In the context of system architecture,central server and the remote control interface are not described in detail. However, the focus is on the design and implementation of the train client using Lingua Franca.
