\section{Conclusion}
\label{sec:conclusion}
    In this paper, we presented a foundational design and implementation of a remote control system for train operations, leveraging the capabilities of WebTransport and the deterministic execution model of Lingua Franca. Our architecture integrates real-time video streaming and low-latency control command transmission, demonstrating the feasibility of using modern web protocols to meet the latency requirements of the remote control system. The modular design of our system allows for scalability and adaptability to various operational scenarios, while the use of Lingua Franca ensures deterministic behavior in real world environments. Our evaluation results from real world tests indicate the feasibility of using WebTransport.

    Future work will focus on several directions. First, we plan to enhance the operator interface by showing situational-awareness features. Second, we intend to conduct larger-scale field tests to evaluate system performance under diverse network conditions. Third, we plan to explore the use of Federated Lingua Franca for distributed execution on multi-train control scenarios.
