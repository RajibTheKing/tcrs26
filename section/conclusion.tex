\section{Conclusion}
\label{sec:conclusion}
    In this paper, we presented a remote control system for real-time model train operation using   WebTransport over QUIC. Our system architecture includes an onboard processor for video streaming and command processing, a central server for communication management, and a web-based remote control console. We evaluated the performance of our system in terms of latency and reliability, demonstrating that WebTransport over QUIC provides low-latency and reliable communication suitable for real-time applications.The experimental results showed that our proposed system achieves satisfactory performance, with low latency and minimal packet loss, making it a viable solution for remote control of model trains. Future work will focus on further optimizing the system, exploring and testing features such as multi-train control, and investigating the applicability of our approach to other real-time remote control scenarios. Overall, our work contributes to the advancement of remote control technologies for real-time systems, leveraging modern web communication protocols to enhance user experience and system performance.
