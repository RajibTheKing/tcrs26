\section{Experiment and Evaluation}
\label{sec:evaluation}
    To evaluate the performance of our remote control system, we conducted a series of experiments under realistic network conditions. The primary focus of our evaluation was to assess the latency and reliability of control command transmission and video streaming using WebTransport in the context of remote train operations. The Lingua Franca–based train client was deployed on a \textbf{Raspberry Pi 5}, which coordinates a physical test vehicle equipped with a camera and a motor control system. Central server were deployed on a cloud instance hosted in Frankfurt, Germany. During the experiments, the physical test vehicle was in Lütjenburg, Germany which was controlled from the laboratory in Kiel University using the Remote Control interface. In these experiments, the train client was connected to the central server via a 5G network using a \textbf{Chateau 5G R17 ax} router. The remote control client was connected to the central server over a wired Ethernet connection.
